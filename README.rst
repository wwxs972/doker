Doker
=====

**Doker** stands for **do**\ cument ma\ **ker**.

Installation
============

Run in console::

  pip install doker

Usage
=====

Create ``mydoc.yaml``::

  entry: .

Create ``main.rst``::

  My First Document
  =================

  **Hello**, _world_!

And run in console::

  doker mydoc

Find ``mydoc.pdf`` in the current directory.
