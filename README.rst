Doker
=====

**Doker** stands for **do**\ cument ma\ **ker**.

Installation
============

Run in console::

  pip install doker

Usage
=====

Create ``minimal.yaml``::

  entry: .

Create ``main.rst``::

  Minimal Document
  ================

  **Hello**, *world*!

And run in console::

  doker --pdf minimal

Find ``minimal.pdf`` in the current directory.

Deeper diving
=============

See `examples <https://github.com/doker-project/doker/tree/master/examples>`__ for most popular use cases.

=============
